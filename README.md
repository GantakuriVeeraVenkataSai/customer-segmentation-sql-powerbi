# Customer Segmentation & Revenue Analysis

## Project Overview
This project analyzes customer purchasing behavior using SQL and Power BI to identify high-value customers and revenue concentration patterns.

## Objectives
- Calculate customer metrics such as Total Spend, Purchase Frequency, and Average Order Value
- Segment customers based on value and purchasing behavior
- Identify revenue concentration using Pareto analysis
- Build an interactive Power BI dashboard for business insights

## Tools Used
- SQL (MySQL)
- Power BI
- Excel Dataset

## SQL Analysis
SQL was used to:

• Calculate Total Spend per customer  
• Calculate Purchase Frequency  
• Calculate Average Order Value  
• Rank customers using Window Functions  
• Identify revenue contribution percentages  

## Key Insights
- High-value customers generate the majority of revenue.
- Customers with higher purchase frequency contribute significantly more revenue.
- Revenue increased steadily from 2016 onwards.
- West region contributes the highest revenue.
